Aby uruchomić aplikację w środowisku lokalnym wpisujemy kolejno w terminalu:

1. Najpierw wpisujemy: 'python -m venv venv'

2. Aktwyujemy środowisko -> Win: '.\venv\Scripts\Activate' -> Mac: 'source venv/bin/activate'

3. Instalujemy biblioteki: 'pip install -r requirements.txt'

4. Uruchamiamy aplikację: 'streamlit run app.py'
